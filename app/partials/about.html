What is TessuStats?

TessuStats is a website / webapp that collects data from the Loungestats Teamspeak server and displays different statistics, graphs and information about how the server is operating. TessuStats is completely non-profit free website and is developed and maintained just for practice, portfolio and own fun. What is TeamSpeak? Check it out here: https://www.teamspeak.com/teamspeak3

The address of Loungestats Teamspeak server is: loungestats.com

Front-end

TessuStats uses Angular.js as the front-end framework. For CSS we use Bootstrap and to render the flags we use flag-icon-css. We also use Moment.js for date and time functions and Google Analytics to see how many visitors we have. For all the graphs we use HighCharts

Back-end

Running the website
The server is running Express.js on top of Node.js. If you want to track your TeamSpeak server the same way, check out the git repository here!
Collecting the data
TessuStats connects to the server via TeamSpeaks official ServerQuery interface.The connection is made using node-teamspeak ServerQuery Node.js library. We make a connection to the TeamSpeak server every 5 minutes. Every scan we save the server state, connection status,channels that are up and search for user list to add a record of the user to the database. We do not store any IP adresses.
Storing the data
We store everything in SQLite database (mostly because I'm lazy to set up a real database). We use sqlite3 Node.js module to access the database.
