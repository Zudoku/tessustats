What is TessuStats?

TessuStats is a website that collects data from the HerGGu Teamspeak server and displays different statistics, graphs and information about how the server is operating. TessuStats is completely non-profit free website and is developed and maintained just for practice, portfolio and own fun.

The address of HerGGu Teamspeak server is: osusearch.com

Frontend

TessuStats uses Angular.js as front end framework to make it a one page application. For CSS we use Bootstrap and to render the flags we use flag-icon-css. We also use Moment.js for date and time functions and Google Analytics to see how many visitors we have. For all the graphs we use HighCharts

Backend

Running the website
The server is running Express.js on top of Node.js. Currently the Git repository for this project is private but I am looking to make it public soon. The web app is not made as custom for the HerGGu server and technically could be deployed to track any Teamspeak server with ServerQuery API open.
Collecting the data
TessuStats connects to the server via TeamSpeaks official ServerQuery interface.The connection is made using node-teamspeak ServerQuery Node.js library. We make a connection to the TeamSpeak server every 5 minutes. Every scan we save the server state, connection status,channels that are up and search for user list to add a record of the user to the database. We do not store any IP adresses.
Storing the data
We store everything in SQLite database. It is serverless so we don't need an SQL server. We use sqlite3 Node.js module to access the database.
